# 江湖行俠錄（WebGL Prototype）

使用 JavaScript + Three.js 製作的瀏覽器 Roguelite 原型，主打低多邊形、俯視等角、快節奏砍殺戰鬥。

## 快速開始

```bash
python3 -m http.server 4173
```

打開 <http://localhost:4173>

## 已實作內容

- 回合制關卡與首領規則：每 3 回合 1 次首領戰。
- 資源條：生命（HP）、內力（MP）、氣（SP）。
  - 每回合開始時 HP/MP 全滿。
  - 撿到丹藥時 HP/MP 全滿。
  - 氣在奔跑與普攻時消耗，且快速自動回復。
- 戰鬥：普攻（J）、三種可購買技能（K/L/:）、衝刺（Shift）。
- 商店：每回合結束後提供技能/工具購買，依銅錢支付。
- 角色介面：武學樹、工具袋、角色屬性三頁。
- 繁體中文 UI 文案。

## 視覺升級（本版）

- 角色改為低多邊形「類人武俠」造型（頭、軀幹、四肢、髮型、武器）。
- 新增動作：
  - 移動時四肢擺動。
  - 普攻時右臂揮砍 + 劍氣弧光。
  - 武學時雙臂施法姿勢 + 旋轉飛彈。
- 場景美術優化：草地紋理、山體、竹林、橋與水池裝飾。
- HUD 與面板採玻璃感半透明樣式。

## 操作

- `WASD`：移動
- `Shift`：衝刺（消耗氣）
- `J`：普攻（消耗氣）
- `K`：震山掌（範圍傷害 + 朝面向發射掌風）
- `L`：金鐘護體（短時間無敵）
- `:`：毒罡護身（接觸你者中毒）
