# 江湖行俠錄（WebGL Prototype）

使用 JavaScript + Three.js 製作的瀏覽器 Roguelite 原型，主打低多邊形、俯視等角、快節奏砍殺戰鬥。

## 快速開始

```bash
python3 -m http.server 4173
```

打開 <http://localhost:4173>

## 已實作內容

- 回合制關卡與首領規則：每 3 回合 1 次首領戰。
- 資源條：生命（HP）、內力（MP）、氣（SP）。
  - 每回合開始時 HP/MP 全滿。
  - 撿到丹藥時 HP/MP 全滿。
  - 氣在奔跑與普攻時消耗，且快速自動回復。
- 戰鬥：普攻（J）、武學飛彈（K）、衝刺（Shift）。
- 商店：每回合結束後提供技能/工具購買，依銅錢支付。
- 角色介面：武學樹、工具袋、角色屬性三頁。
- 繁體中文 UI 文案。

## 操作

- `WASD`：移動
- `Shift`：衝刺（消耗氣）
- `J`：普攻（消耗氣）
- `K`：武學（消耗內力）
